# JLSQUANT
Jorge Martínez Romeral version of LSQUANT based on python and including some BLAS functions in C++

## Setup instructions
- Install a conda enviroment:

	wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh  
	bash Miniconda3-latest-Linux-x86_64.sh  
	source ~/.bashrc  
	conda init  
	conda --version  


- Install required packages with following versions :

	pip install numpy==1.26.3  
	pip install scipy==1.12.0  
	pip install cython==3.0.11  
	
- Installing git with ssh key :

	git config --global user.name "jorgemr42"  
	git config --global user.email "jorgecoslada@gmail.com"  
	ls -al ~/.ssh  
	ssh-keygen -t rsa -b 4096 -C "jorgecoslada@gmail.com"  
	eval "$(ssh-agent -s)"  
	ssh-add ~/.ssh/id_rsa  
	cat ~/.ssh/id_rsa.pub  
	ssh -T git@github.com  
	git clone git@github.com:jorgemr42/JCLSQUANT.git  

- Installing CUDA packages where obv is the name of the .cu file:  
	nvcc -Xcompiler -fPIC -shared -o libobv.so obv.cu -arch=sm_89  
	cp ./libobv.so ../lib/  
	cp ./libobv.so ../../  

- Installing Cython packages:

	python name.py build_ext --inplace  

- Installing all:  

	pip install -e .  

- Clonning: 
	git clone -b main git@github.com:jorgemr42/JCLSQUANT.git
